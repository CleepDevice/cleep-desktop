build Install Cleep image on SD card manually
label You can install Cleep manually downloading image file directly from Cleep website.
label Once downloaded, burn image with your favorite SD card burner application or use Etcher (Cross plaftorm software adviced by Raspberry Pi fundation).
label Plug the SD card in your Raspberry Pi and connect it to power supply.
label Wait few seconds and you should see a new device appear in the left "Devices" panel.
