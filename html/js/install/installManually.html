Install Cleep manually on your device

build Install Cleep image on SD card manually
You can install Cleep manually downloading image file directly from Cleep website.
Once downloaded, burn image with your favorite SD card burner application or use Etcher (Cross plaftorm software adviced by Raspberry Pi fundation).
Plug the SD card in your Raspberry Pi and connect it to power supply.
Wait few seconds and you should see a new device appear in the left "Devices" panel.
Install Cleep package on existing operating system
Cleep is running exclusively on Raspbian operating system.
Connect to your Raspberry Pi via SSH
Download Raspiot package from Cleep website: wget http://www.cleep.me/downloads/raspiot_X.X.X_armhf.deb
Install tools: aptitude update; aptitude install ?
Install dependencies: aptitude install ?
And install Cleep package: dpkg -i raspiot_X.X.X_armhf.deb
