Welcome to CleepDesktop!

CleepDesktop

The application that allows you to install, detect and configure your Cleep devices as simply as possible.
This content explains you how to use CleepDesktop. It is also available on help menu.

Application window

Left panel

The left panel is the devices panel. It allows you to select a device to access its configuration.

Cleep devices are automatically discovered by CleepDesktop as soon as they have started.

You will find different sections in this devices panel:
  • An unconfigured device section that shows you your new devices ready to be configured
  • A configured devices section that displays your configured devices with their name you give them


On the left of each devices, you will see an icon that represents device status. Icons means:
  • for device that is not accessible (powered off or disconnected from network)
  • for online device


Clicking on device icon will display device informations like installed apps and hardware details.

Right panel

The right panel is the dynamic one. It will display different application pages such as:
  • Application updates
  • Device pages
  • Device installation
  • Help infos (like this page)
  • ...
 Toolbar
Toolbar is the leftmost area of the application. It contains quick access buttons to application features
Cleep installation
This page is dedicated to Cleep installation on your device.

It allows you to prepare easily a SD card helping you:
  • to choose the latest Cleep software available,
  • to select the appropriate drive for the installation,
  • to configure the device wifi connection in case you need it
Finally you can follow full installation progress (download, installation and final checks)

In case of trouble you will find a link to manual Cleep installation.
Monitoring
The monitoring page allows you to follow messages sent by your Cleep devices.
All messages are not distributed by a device, only useful messages such as temperature, motion, weather...
Parameters
This is a quick access to CleepDesktop preferences. Preferences can also be opened using main application menu.
Application menu
Updates
The updates page helps you following available software updates.

Please note, Etcher is the 3rd part software used to install Cleep software on SD card.
Preferences
Preferences page allows you to configure CleepDesktop application.
