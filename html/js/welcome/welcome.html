Welcome

Welcome to CleepDesktop!

CleepDesktop is an application that allows you to detect, configure and monitor your Cleep devices easily.

Main window

Left panel

The left panel is the devices panel. It allows you to select a device to access its configuration.

Cleep devices are automatically discovered by CleepDesktop as soon as they have started.

You will find different sections in this devices panel:
  • An unconfigured device section that shows you your new devices ready to be configured
  • A devices section that displays your configured devices with their name you give them


On the left of each devices, you will see an icon that represents device status. Icons signification:
  • for device that is not accessible (ie powered off or disconnected from network)
  • for online device

Right panel

All different application views are displayed in the right panel.
In this panel will be displayed:
  • Application preferences
  • Device configuration pages
  • Cleep installation pages
  • ...

Cleep software installation

Cleep install menu gives you access to different way to install Cleep on your Raspberry pi:
  • Easy install menu guides you to install easily Cleep software on your Raspberry pi SD card directly from CleepDesktop.
  • While Manual install menu will explain you all steps to install manually Cleep software on existing installation.

Need help ?

You are lost or need some help, you can post messages on Cleep forum directly accessible from help menu.

You found a bug? You have quick access to Cleep tracking bugs page directly from help menu.
